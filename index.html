Info acerca de la web

Esta es mi primera web cuyo objetivo es lograr que aquellas personas que quieren adoptar su primer perro logren hacerlo de una manera idonea. Para lograr este cometido se le facilitara a el usuario el contacto con refugios así como tambien se le brindará información de gran utilidad a la hora de adoptar un primer animal así como también para aquellos que ya tengan animales.
